# 0090 — Binary Search Tree

Insert `5, 3, 8, 1, 4` into a binary search tree and print an in-order traversal (which yields the values in sorted order): `1 3 4 5 8`. `insert` recurses left for smaller values and right otherwise; an in-order traversal (left, node, right) visits a BST's values in sorted order.

## Run

    python3 main.py
