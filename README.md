# python-lang

Incremental lessons learning Python.

## Prerequisites

- Python 3.x

### Install

- macOS (Homebrew): `brew install python`
- Debian / Ubuntu: `sudo apt install python3`
- Fedora: `sudo dnf install python3`
- Other: see [python.org/downloads](https://www.python.org/downloads/)

### Verify

    python3 --version
