# 0034 — Default & Named Arguments

Give a `greet` function a default greeting, then call it once without the greeting and once overriding it, printing `Hello, Ada` and `Hi, Ada`. A parameter with `= "Hello"` supplies a default when the argument is omitted, and any argument can be passed by name (`greeting="Hi"`). Keyword arguments may be given in any order after the positionals.

## Run

    python3 main.py
