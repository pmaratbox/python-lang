# 0012 — Enums

Define a `Color` enum (`RED`, `GREEN`, `BLUE`) and print the integer value of
`GREEN` (1) and `BLUE` (2). `IntEnum` members are named constants that *are*
integers, so they compare and print as their value. A plain `Enum` instead gives
opaque members whose `.value` you read explicitly.

## Run

    python3 main.py
