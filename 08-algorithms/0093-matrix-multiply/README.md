# 0093 — Matrix Multiply

Multiply the 2x2 matrices `[[1, 2], [3, 4]]` and `[[5, 6], [7, 8]]` and print the product, one row per line: `19 22` and `43 50`. Each entry `result[i][j]` is the dot product of row `i` of `a` and column `j` of `b`, accumulated by the inner `k` loop.

## Run

    python3 main.py
