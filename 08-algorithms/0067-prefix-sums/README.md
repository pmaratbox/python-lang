# 0067 — Prefix Sums

Compute the running totals (prefix sums) of `1, 2, 3, 4` — each element added to the sum of all the previous ones — and print them: `1 3 6 10`. `itertools.accumulate` yields each running total (summing with `+` by default), turning `1, 2, 3, 4` into `1, 3, 6, 10`.

## Run

    python3 main.py
