# 0046 — Number Bases & Radix

Print the integer `255` in hexadecimal, octal, and binary, producing `hex: ff`, `oct: 377`, and `bin: 11111111`. An f-string format spec selects the base: `{n:x}` is lowercase hex, `{n:o}` octal, and `{n:b}` binary — no prefix is added. Use `0x`/`0o`/`0b` literals to write numbers in those bases.

## Run

    python3 main.py
