# 0025 — Tuples & Multiple Return

Return both the smaller and larger of `3` and `7` from one function, unpacking the pair to print `min: 3` and `max: 7`. A function returns a `tuple` simply by listing values; `(min(a, b), max(a, b))` packs them and `lo, hi = ...` unpacks by position. Tuples are immutable and the surrounding parentheses are optional.
