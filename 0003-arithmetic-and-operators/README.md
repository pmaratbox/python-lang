# 0003 — Arithmetic & operators

Compute and print the five elementary arithmetic operations on two integers
(`a = 17`, `b = 5`). In Python, `/` is **true division** and always returns a
`float` (`17 / 5 → 3.4`); use `//` for integer (floor) division and `%` for
modulo.

## Run

    python3 main.py
