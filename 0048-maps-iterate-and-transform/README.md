# 0048 — Maps: Iterate & Transform

Build a map from letters to numbers (`a`->1, `b`->2, `c`->3), sum all its values, and print `sum: 6`. `dict.values()` is a view over the values, and the built-in `sum` folds them. `.keys()`, `.items()`, and iterating the dict directly (which yields keys) are the other traversal options.

## Run

    python3 main.py
