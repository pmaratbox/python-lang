# 0009 — Maps

Build a dictionary `{"one": 1, "two": 2, "three": 3}`, look up `"two"`, and
print its value and the dictionary's size. `dict` is Python's built-in hash
map; `d[key]` looks up a value (raising `KeyError` if absent — use `d.get(key)`
for a `None` default), and `len(d)` counts the entries.

## Run

    python3 main.py
