# 0561 — SHA-1

Compute the SHA-1 digest of the string `hello` using Python's standard-library `hashlib` module and print it as a lowercase hex string. SHA-1 is a legacy 160-bit hash and is no longer considered secure for cryptographic use, but it remains useful for learning and for non-security checksums. `hashlib.sha1(...).hexdigest()` returns the digest deterministically.

## Run

    python3 main.py
