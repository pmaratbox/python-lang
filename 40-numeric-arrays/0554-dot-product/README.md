# 0554 — Dot product

Computes the dot product of two vectors with NumPy. The vectors `[1, 2, 3]` and `[4, 5, 6]` are built as `np.array`, and the `@` operator (`np.dot`) contracts them into the scalar `1*4 + 2*5 + 3*6 = 32`, cast to an integer for byte-identical output.

## Run

    python3 main.py
