# 0007 — Collections

Build a list of the integers `1, 2, 3, 4, 5`, then print its count and its
first and last elements. `[...]` is a list (dynamic and heterogeneous); `len()`
is a builtin function, not a method. Negative indices count from the end, so
`nums[-1]` is the idiomatic "last element".

## Run

    python3 main.py
