# 0049 — Zip & Enumerate

Pair the letters `a, b, c` with the numbers `1, 2, 3` position by position, formatting each pair as `key=value` and printing `a=1 b=2 c=3`. `zip(letters, nums)` yields pairs position by position, stopping at the shorter input; unpacking `(k, n)` in a comprehension builds the strings. Its enumerate counterpart, `enumerate`, pairs items with their index.

## Run

    python3 main.py
