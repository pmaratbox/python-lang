# 0008 — Strings

Given `name = "world"`, print a greeting, the name in uppercase, and its
length. f-strings (`f"Hello, {name}!"`) interpolate; `.upper()` returns a new
uppercased string (strings are immutable); `len()` counts characters. Python 3
strings are sequences of Unicode code points, so `len()` is a character count,
not a byte count.

## Run

    python3 main.py
