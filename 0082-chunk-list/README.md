# 0082 — Chunk a List

Split the list `1, 2, 3, 4, 5, 6, 7` into chunks of `3` and print each chunk on its own line: `1 2 3`, `4 5 6`, `7`. Stepping the range by `size` and slicing `nums[i:i+size]` yields successive chunks; the final slice is short when the length is not a multiple.

## Run

    python3 main.py
