# 0060 ‚Äî Priority Queue

Push `3`, `1`, and `2` into a min-priority-queue, then pop them all and print them in priority (ascending) order: `1 2 3`. `heapq` turns a plain list into a binary min-heap: `heappush` and `heappop` keep the smallest element at the front in O(log n).

## Run

    python3 main.py
