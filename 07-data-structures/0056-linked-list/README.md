# 0056 — Linked List

Build a singly-linked list holding `1`, `2`, and `3`, then traverse it from head to tail and print `1 -> 2 -> 3`. Each `Node` holds a `value` and a reference to the `next` node (or `None` at the tail); traversal follows the references until `None`. Python objects are references, so this is a genuine linked structure.

## Run

    python3 main.py
