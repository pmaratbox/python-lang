# 0051 — Ternary & Conditional Expressions

For each of `4` and `7`, choose `even` or `odd` with a single conditional expression and print `4: even` and `7: odd`. Python's conditional expression reads value-first: `"even" if n % 2 == 0 else "odd"`. There is no `?:` operator — this inline `if`/`else` is the equivalent.

## Run

    python3 main.py
